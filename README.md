# 😊 Sentiment Analysis using Machine Learning

A Machine Learning-based **Sentiment Analysis** application that predicts whether a piece of text expresses **Positive**, **Negative**, or **Neutral** sentiment.

The application performs data preprocessing, feature extraction, model training, evaluation, and real-time sentiment prediction through an interactive web interface.

---

## 📌 Features

- 🔍 Sentiment Classification
- 🧹 Text Cleaning & Preprocessing
- 🤖 Machine Learning Model Training
- 📊 Model Evaluation
- 🌐 Interactive Web Interface
- 💾 Saved Trained Models
- ⚡ Real-time Predictions
- 📈 Performance Metrics

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Shrutisinha/sentimental_analysis.git
```

## Move to Project Folder

```bash
cd sentimental_analysis
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Project

### Flask

```bash
python app.py
```

### Streamlit

```bash
streamlit run app.py
```

### Model Only

```bash
python main.py
```
